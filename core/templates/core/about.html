About this project

A beginner-friendly Django project polished with practical conventions.

AI Site combines a small article section, a product market, user authentication, profiles, and a contact form. The structure stays intentionally simple so it is easy to read, extend, and discuss in a portfolio.

It uses Django templates, Bootstrap 5, model forms, session-based cart storage, and the built-in authentication system.
